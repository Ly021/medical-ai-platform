NestJS 是一个用于构建高效、可扩展的 Node.js 服务器端应用程序的框架。它使用渐进式 JavaScript，内置并完全支持 TypeScript。

NestJS 的核心概念包括：
- 模块（Module）：组织代码的基本单元，使用 @Module 装饰器
- 控制器（Controller）：处理 HTTP 请求，定义路由
- 服务（Service）：封装业务逻辑，可被注入到控制器中
- 中间件（Middleware）：在请求处理前后执行
- 守卫（Guard）：处理权限验证
- 拦截器（Interceptor）：转换请求和响应
- 管道（Pipe）：数据验证和转换
- 过滤器（Filter）：异常处理

NestJS 使用依赖注入（DI）容器管理所有可注入的对象。每个 provider 默认是单例的。

LangChain 是一个用于构建 LLM 应用的开源框架，支持 Python 和 JavaScript。它提供：
- 模型集成：统一接口调用各种大模型
- 工具（Tool）：让模型调用外部函数
- Agent：让模型根据用户输入自主决定调用哪些工具
- RAG（检索增强生成）：从知识库检索文档辅助回答
- 向量存储：存储和检索文档嵌入

RAG 的典型流程是：
1. 文档加载：读取本地或远程文档
2. 文本分割：将长文档切成小块
3. 向量化（Embedding）：将文本块转换为向量
4. 存储到向量数据库
5. 检索：用户提问时搜索最相关的文本块
6. 增强生成：将检索结果拼入 prompt 发给模型

Agent 的工具调用流程：
1. 用户发送消息
2. 模型分析意图，决定是否调用工具
3. 如果需要，模型生成 tool call（工具名 + 参数）
4. 系统执行工具函数，获取结果
5. 将工具结果返回给模型
6. 模型综合工具结果生成最终回复
